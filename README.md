# Shopping Discount Calculator 🛒

A lightweight JavaScript application designed to demonstrate the core concepts of **Operators** and **Expressions** in web development. This project calculates the final bill for a product by applying taxes and conditional discounts.

## 🚀 Features
- **Arithmetic Calculations:** Automates tax and total price calculation.
- **Dynamic Logic:** Uses relational and logical operators to verify discount eligibility.
- **Type Safety:** Implements strict equality checks (`===`) for robust logic.

## 🛠️ Technologies Used
- **HTML5:** Basic structure for the project.
- **JavaScript (ES6):** Core logic, operators, and expressions.

## 📖 How It Works
The program follows these logical steps:
1. **Input:** Sets the base price, tax rate, and customer status.
2. **Process:** 
   - Calculates tax using `*`.
   - Checks if the total is over $500 **AND** the user is a VIP using `&&`.
3. **Output:** Applies a flat discount if eligible and logs the final breakdown to the console.

## 💻 How to Run
1. Clone this repository or download the files.
2. Open `index.html` in any modern web browser.
3. Open the **Developer Tools** (Right-click > Inspect > Console) to see the output.

## 📝 Key Concepts Learned
- **Arithmetic Operators:** `+`, `*`, `-`
- **Relational Operators:** `>`
- **Logical Operators:** `&&`
- **Expressions:** Combining variables and symbols to resolve a final value.
